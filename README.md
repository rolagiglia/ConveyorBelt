# ConveyorBelt
ConveyorBelt – Sistema de Control de Cinta Transportadora con ESP32. Proyecto académico de IoT basado en ESP32 que controla una cinta transportadora mediante sensores, actuadores y lógica programada en C/C++. Incluye control de motor PWM, lectura de sensores, detección de obstáculos y funciones básicas de seguridad.
